Project 01

NetFlix clone

The Netflix clone project was an incredibly enriching experience that allowed me to apply my HTML and CSS skills practically. During this three-week project, I aimed to design a visually captivating and organized interface, similar to the original Netflix layout. First, I created a well-structured HTML framework that efficiently implemented semantic elements, giving meaning to the content and improving SEO aspects. With CSS, I worked on creating a responsive design that would be functional across various devices. I utilized Flexbox and Grid systems for layout, which enabled me to design a clean UI with a seamless transition of elements. My focus was on ensuring that the homepage showcased featured movies in a visually appealing manner, utilizing hover effects to make the experience interactive and engaging. I experimented with advanced styling techniques such as animations and transitions, which further enhanced the visual appeal of the website.
